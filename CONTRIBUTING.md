# Contributing

Thanks for helping improve Revopoint Turntable for Home Assistant.

## Testing

Please include the Home Assistant version, turntable model, Bluetooth adapter or proxy model, and relevant log entries when reporting issues.

Large Turntable / TA500 support is experimental because the maintainer does not have that hardware. Reports from users with that model are especially helpful.

## Development

- Keep the integration local-only and cloud-free.
- Do not commit secrets, Home Assistant tokens, generated archives, diagnostic dumps, or local cache files.
- Run Python syntax checks before submitting changes.
- Keep user-facing docs focused on installation, supported devices, and behavior.

## Issues

When opening an issue, include:

- Home Assistant version.
- Integration version.
- Turntable model.
- Whether you are using a local Bluetooth adapter or Bluetooth proxy.
- A short description of the expected and actual behavior.
- Any relevant log entries with personal information removed.
